Detailed Description

Represents the result of an in-app billing operation. A result is composed of a response code (an integer) and possibly a message (String). You can get those by calling getResponse and getMessage(), respectively. You can also inquire whether a result is a success or a failure by calling isSuccess() and isFailure().
